Задание №3
Чему будет равно JS-выражение 1000 + "108"?

Выражение будет равно «1000108». Т.к. при сложении JS будет приводить правую и левую часть к одинаковому типу, а так как они не одинаковы, то всё будет приведено к типу « строка».

Задание №3
Самостоятельно разобраться с атрибутами тега script (async и defer)

async — скрипты грузятся асинхронно. первым загружается скрипт, который меньше всех весит.

defer — скрипты грузятся синхронно в порядке их указания в файле html.
